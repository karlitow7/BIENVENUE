site web Vanill'île

Dans le cadre d'un travail de groupe, nous avons conçu et développé le site vitrine Vanill'île, dédié à une marque fictive de bonbons bio à base de vanille Pompona.

Ce site présente les produits, l'identité de marque, les valeurs écologiques, les partenaires potentiels et les éléments de stratégie de communication.

SAE 201 - mettre en pratique les bases du développement web, comprendre la structure d'un site statique, et de travailler en collaboration sur un projet concret.

Création complète d'un site internet portfolio codé en HTML, CSS et JavaScript. Ce projet regroupe l'ensemble de mes travaux réalisés durant mes 3 années de MMI ainsi que mes projets musicaux personnels.
